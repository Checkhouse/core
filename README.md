## add product 중고 상품

- 중고 상품 등록
    - 중고 상품을 등록합니다.
- 중고 상품 등록 상태 변경(등록 중, 판매 중, 판매 완료)
    - 중고 상품의 등록 상태를 변경합니다.
- 중고 상품 정보 조회
    - 특정 중고 상품의 정보를 조회합니다.
- 등록된 중고 상품 등록 취소
    - 중고 물품 상품을 취소합니다.
- 특정 상태 중고 상품 리스트
    - 특정 원본 상품의 특정 상태 중고 상품 리스트를 조회합니다.
- 특정 사용자가 등록한 중고 상품 리스트 조회

## pickup 픽업

- 특정 스토어의 픽업 거래 리스트 조회
